# kgnas

基于 Python 标准库的轻量 NAS（网络附加存储）系统。  
无需安装任何第三方依赖，提供基本的文件管理、上传下载、在线预览等功能。

## 功能

- 密码保护（首次启动时设置）
- 文件列表浏览，支持排序（名称、大小、修改时间）
- 进入子目录 / 返回上级目录
- 上传文件（支持多选、拖拽，含文件夹上传）
- 下载文件（支持单个或批量）
- 文件操作：删除、重命名、复制、剪切、粘贴
- 新建文件夹
- 在线预览：图片、文本文件、音视频（MP3/MP4 等）
- 显示存储空间使用情况（总容量、已用、剩余）
- 显示本机 IPv4 / IPv6 地址
- 显示内存使用情况（Windows 下）
- 修改 Web 服务端口（需重启生效）
- 修改登录密码
- 从网页关闭服务器或关闭宿主机（需操作系统权限）

## 运行环境

- Python 3.6 或更高版本
- 操作系统：Windows / Linux / macOS

## 快速开始

1. 将 `kgnas_server.py` 与 `index.html` 放在同一个目录中。
2. 打开终端（cmd / PowerShell / bash），进入该目录。
3. 执行：

```bash
python kgnas_server.py
浏览器访问 http://127.0.0.1:8886 或 http://<本机IP>:8886。

首次访问时会引导设置登录密码和文件存储路径。
存储路径需已存在且具有读写权限（程序不会自动创建父目录）。

配置说明
配置文件 kgnas_config.json 与脚本同目录，内容示例：

json
{
  "password": "用户设置的密码（明文存储）",
  "storage_path": "D:\\NAS_Storage",
  "port": 8886,
  "initialized": true
}
修改端口后需重启 kgnas_server.py 生效。

密码修改可通过网页「系统设置」完成。

使用提示
文件管理页面支持多选（勾选复选框），可批量删除/复制/剪切/下载。

上传大文件时受限于 HTTP 请求超时和内存，建议单文件不超过 2GB。

批量下载时浏览器可能会拦截多个弹窗，请允许弹出窗口。

若需外网访问，请自行配置端口转发或内网穿透，并确保密码强度。

技术栈
后端：Python 3 + http.server + socketserver（均属标准库）

前端：原生 HTML / CSS / JavaScript（无框架）

通信：REST API（JSON）+ multipart/form-data（上传）
